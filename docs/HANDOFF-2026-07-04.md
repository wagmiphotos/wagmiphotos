## What shipped

### 1. Library page (`#/library`)
Public keyword search + download over the shared image library.
- `GET /v1/library?q=&limit=&offset=` ‚Äî D1 `LIKE` over `assets.prompt`,
  whitespace-tokenized AND terms, `has_more` pagination. Migration
  `0003_assets_browse_index.sql` backs the browse ordering.
- `GET /v1/library/:id/download` ‚Äî CORS-safe worker proxy, attachment filename
  from prompt slug; 404 on unknown/malformed id.
- `src/library.ts` (handlers), routed in `src/index.ts`; `#/library` SPA view
  (debounced search, responsive grid, lazy thumbs, load-more).
- Also fixed en route: playground/docs no longer reference the retired
  `shared-cache-*` "target model" ‚Äî the API is `prompt` + `cache_tolerance` +
  `generate_on_miss`, and the playground now has a generate-on-demand toggle.

### 2. Configurable public URLs
- `wrangler.toml` vars `PUBLIC_SITE_URL` / `PUBLIC_API_BASE_URL`
  (`wagmi.photos` / `api.wagmi.photos/v1`). `src/rewrite.ts` substitutes them
  into served HTML at request time, so a dev deployment renders its own URLs.
  SPA fetches stay origin-relative (no CORS). Verified E2E with a `.dev.vars`
  override on a second wrangler instance.

### 3. GMI embedder + backfill deployment
The missing CLIP endpoint + a production home for the backfill.
- `projects/embedder/` ‚Äî FastAPI CLIP ViT-L/14 service; protocol matches the
  existing Worker (`src/embed.ts`) and Python (`common/clip.py`) clients;
  tests offline/torch-free (model deps behind a `[model]` extra).
- `deploy/gmi/` ‚Äî compose stack: `embedder` + `backfill` + `cloudflared`
  tunnel publishing `embed.wagmi.photos`. Image builds with weights baked in;
  smoke-tested serving real 768-dim vectors.
- Hardened per review: **fail-closed auth** (container refuses to boot without
  `EMBED_TOKEN` ‚Äî it's public via the tunnel), pinned model deps
  (torch 2.12.1 / torchvision 0.27.1 / open-clip 3.3.0 / pillow 12.2.0) +
  `cloudflared:2026.6.1`, inference moved off the event loop, 10 MB body cap.
